Qué cambia respecto a TriagerX

ComponenteTriagerXtriager-omega
CBR Ensemble de 2 transformers clasificadores (RoBERTa + DeBERTa), entrenados Recuperación de casos (kNN + voto al resolvedor) con encoder congelado, sin entrenar; simple e interpretable
Texto Título + descripción + destilación con LLM (vistas raw / distilled / both)
IBR Interacciones de GitHub (commits / PRs) Interacciones tipadas (contribution / assignment / discussion) con pesos IP + decaimiento temporal
Fusión FS = NPS + W_f·NIS (W_f fijo ≈ 0.7) Misma fórmula; W_f depende del régimen de datos (hallazgo)
Hallazgos — El recuperador iguala su ensemble sin entrenar (OpenJ9 0.2715 vs 0.270) y gana en cola larga; el IBR aporta solo cuando es fuerte (Mozilla sí, OpenJ9-50 no)
